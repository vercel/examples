Live Example: https://example-hono-bun.vercel.app/

Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

To develop locally:

```
bun install
vc dev
```

```
open http://localhost:3000
```

To build locally:

```
bun install
vc build
```

To deploy:

```
bun install
vc deploy
```
